
Ülesanne 1: Luua toodete raport terminalis, mis näitab laoseisu, hinnangut, soodustust, specs.
Ülesanne 2: Luua interaktiivne veebirakendus, milles on toodete list, saab lisada tooteid, filter ja sort on olemas, LocalStorages salvestamine.

Käivitamine
Initseerimine: ```npm iinit -y ```
Paigalda sõltuvused: ```npm i ```
Kompileeri TypeScript: ```npx tsc -p tsconfig.json ```
Käivita Task 1: ```node dist/task1.js ```. Kui tekkib viga Failed to load the ES module... Make sure to set "type": "module" tuleb vahetada "type":commonJS vahetada "type: module vastu.

Käivita Task 2: ava `src/task2.html`
